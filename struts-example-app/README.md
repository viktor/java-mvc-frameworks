

### Commands to use

```
To create project template
mvn archetype:generate -DgroupId=com.strutsexample -DartifactId=struts-example-app -DarchetypeArtifactId=maven-archetype-webapp -DinteractiveMode=false

To create java folder
mkdir src/main/java

To Compile and run
$ mvn clean package && mvn tomcat7:run

To get dependencies in eclipse
mvn eclipse:eclipse -Dwtpversion=1.5
```
